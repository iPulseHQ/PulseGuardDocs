# PulseQR - Dynamische QR Code Generator

## Overzicht

**PulseQR** is een geavanceerde QR code generatie service die volledig geïntegreerd is met PulseGuard. Het systeem genereert dynamische QR codes die gebruikt kunnen worden voor status pagina's, incident communicatie, en andere monitoring-gerelateerde toepassingen.

## Belangrijkste Features

### Dynamische QR Codes
- **Real-time Updates**: QR codes die automatisch updaten met nieuwste status
- **Custom Branding**: Volledig customizable design en branding
- **Multiple Formats**: Ondersteuning voor verschillende QR code standaarden
- **Error Correction**: Betrouwbare foutcorrectie voor scanning

### PulseGuard Integratie
- **Status Page QR**: Directe links naar actuele status informatie
- **Incident QR**: QR codes die naar incident details leiden
- **Maintenance QR**: Automatische QR codes voor onderhoudsvensters
- **API Gebaseerd**: Volledige integratie via REST API

### Analytics & Tracking
- **Scan Tracking**: Monitor hoeveel mensen QR codes scannen
- **Geographic Data**: Zie waar scans vandaan komen
- **Device Analytics**: Inzicht in gebruikte apparaten en browsers
- **Conversion Tracking**: Meet effectiviteit van QR code campagnes

### Customization Opties
- **Logo Integration**: Voeg bedrijfslogo toe aan QR codes
- **Color Schemes**: Custom kleuren voor branding
- **Size Options**: Verschillende formaten voor diverse toepassingen
- **Background Patterns**: Optionele achtergrond patronen

## Technische Specificaties

### QR Code Standaarden
- **QR Code 2005**: Basis standaard met backward compatibility
- **Micro QR**: Compacte versie voor kleine oppervlaktes
- **iQR Code**: Vierkante versie voor betere error correction
- **Frame QR**: Ronde versie voor speciale toepassingen

### Error Correction Levels
- **Level L**: 7% error correction (snelste)
- **Level M**: 15% error correction (balanced)
- **Level Q**: 25% error correction (betrouwbaar)
- **Level H**: 30% error correction (maximaal betrouwbaar)

### Ondersteunde Content Types
- **URL**: Web links naar status pagina's
- **Text**: Plaine tekst berichten
- **Email**: Directe email links
- **Phone**: Telefoonnummer dialing
- **SMS**: SMS berichten
- **Location**: GPS coordinaten
- **WiFi**: WiFi netwerk configuratie

## API Specificaties

### QR Code Generatie
```bash
# Genereer nieuwe QR code
POST /api/pulseqr/generate
Content-Type: application/json
{
  "content": {
    "type": "url",
    "value": "https://status.company.com"
  },
  "design": {
    "size": 300,
    "error_correction": "M",
    "logo": "https://company.com/logo.png"
  },
  "metadata": {
    "name": "Status Page QR",
    "tags": ["status", "public"],
    "expires_at": "2024-12-31T23:59:59Z"
  }
}

# Response
{
  "qr_id": "qr-123456",
  "qr_code_url": "https://qr.pulseqr.com/qr-123456.png",
  "short_url": "https://pq.rs/abc123",
  "expires_at": "2024-12-31T23:59:59Z"
}
```

### QR Code Management
```bash
# Haal QR code details op
GET /api/pulseqr/{qr_id}

# Update QR code
PUT /api/pulseqr/{qr_id}
Content-Type: application/json
{
  "content": {
    "value": "https://status.company.com/incident-123"
  }
}

# Verwijder QR code
DELETE /api/pulseqr/{qr_id}

# Haal scan analytics op
GET /api/pulseqr/{qr_id}/analytics?period=30d
```

### Batch Operaties
```bash
# Genereer multiple QR codes
POST /api/pulseqr/batch/generate
Content-Type: application/json
{
  "qrs": [
    {
```

## Integratie Voorbeelden

### PulseGuard Status Page Integratie
```javascript
// Automatische QR code generatie bij incident
const incidentCreated = async (incident) => {
  const qrCode = await fetch('/api/pulseqr/generate', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${PULSEGUARD_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      content: {
        type: 'url',
        value: `https://status.company.com/incidents/${incident.id}`
      },
      design: {
        logo: 'https://company.com/logo.png',
        primary_color: '#ff4444'
      },
      metadata: {
        name: `Incident ${incident.id} QR`,
        tags: ['incident', 'auto-generated']
      }
    })
  });

  const qrData = await qrCode.json();
  // Gebruik qrData.qr_code_url in notificaties
};
```

### Mobile App Integratie
```swift
// iOS Swift voorbeeld
func generateStatusQR() async throws -> UIImage {
    let qrRequest = QRGenerationRequest(
        content: .url(URL(string: "https://status.company.com")!),
        design: QRDesign(
            size: 300,
            errorCorrection: .medium,
            logo: UIImage(named: "company-logo")
        )
    )

    let qrImage = try await PulseQRService.shared.generateQR(request: qrRequest)
    return qrImage
}
```

## Troubleshooting

### Veelvoorkomende Problemen

#### QR Code Scant Niet
- Controleer error correction level (gebruik H voor drukwerk)
- Verificeer contrast tussen voor- en achtergrond
- Test met multiple QR code readers
- Controleer of URL correct is

#### Analytics Data Ontbreekt
- Controleer of QR code recentelijk gegenereerd is
- Verificeer API key permissions
- Controleer firewall settings voor analytics endpoints

#### Custom Logo Problemen
- Logo moet minimaal 100x100 pixels zijn
- Gebruik PNG of SVG format
- Zorg voor voldoende contrast
- Test logo op verschillende formaten

### Debug Tools
```bash
# Test QR code generatie
curl -X POST "https://api.pulseqr.com/api/generate" \
  -H "Authorization: Bearer YOUR_API_KEY" \

# Controleer QR code status
curl -X GET "https://api.pulseqr.com/api/{qr_id}" \
  -H "Authorization: Bearer YOUR_API_KEY"

# Haal debug informatie op
curl -X GET "https://api.pulseqr.com/api/{qr_id}/debug" \
  -H "Authorization: Bearer YOUR_API_KEY"
```
