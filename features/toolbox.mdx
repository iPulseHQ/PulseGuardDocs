## Overview

De Toolbox biedt diverse utilities voor het diagnosticeren en analyseren van je web resources. Deze tools helpen je bij het oplossen van problemen, optimaliseren van performance en waarborgen van security compliance.

<Note>
  Volledige toegang tot alle toolbox features is beschikbaar op Pro en Expert plannen. Gratis accounts hebben beperkte toegang tot bepaalde tools.
</Note>

## DNS Lookup Tool

De DNS Lookup tool stelt je in staat om verschillende DNS record types voor elk domein te controleren:

<Steps>
  <Step title="Navigeer naar de Toolbox">
    Klik op "Toolbox" in de hoofdnavigatie sidebar
  </Step>
  <Step title="Selecteer DNS Lookup">
    Klik op de DNS Lookup tool card
  </Step>
  <Step title="Voer domeinnaam in">
    Voer het domein in dat je wilt controleren (bijv. example.com)
  </Step>
  <Step title="Selecteer record types">
    Kies welke DNS record types je wilt opvragen (of selecteer "All")
  </Step>
  <Step title="Bekijk resultaten">
    Controleer de DNS records in het resultaten paneel
  </Step>
</Steps>

## Port Scanner Tool

De Port Scanner helpt je bij het identificeren van open poorten en potentiële beveiligingsrisico's:

<Steps>
  <Step title="Navigeer naar Port Scanner">
    Selecteer de Port Scanner tool uit de toolbox
  </Step>
  <Step title="Voer target in">
    Voer een hostname of IP adres in
  </Step>
  <Step title="Configureer scan">
    Kies scan type (quick, full, custom), protocol (TCP/UDP), en timeout
  </Step>
  <Step title="Start scan">
    Klik op "Start Scan" om te beginnen
  </Step>
  <Step title="Analyseer resultaten">
    Bekijk open poorten, services en potentiële risico's
  </Step>
</Steps>

## IP Geolocation Tool

De IP Geolocation tool geeft gedetailleerde informatie over IP adressen:

<Steps>
  <Step title="Navigeer naar IP Geolocation">
    Selecteer de IP Geolocation tool
  </Step>
  <Step title="Voer IP adres in">
    Voer een IPv4 of IPv6 adres in
  </Step>
  <Step title="Haal informatie op">
    Klik op "Lookup" voor geografische en ISP informatie
  </Step>
  <Step title="Analyseer data">
    Bekijk locatie, ISP, organisatie en andere metadata
  </Step>
</Steps>

## API Gebruik

Alle toolbox tools zijn ook beschikbaar via de REST API:

```bash
# DNS Lookup via API
curl -X POST "https://api.ipulse.one/toolbox/dns-lookup" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"domain": "example.com", "record_type": "A"}'

# IP Geolocation via API
curl -X POST "https://api.ipulse.one/toolbox/ip-scanner" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"ip": "8.8.8.8"}'

# Port Scan via API (streaming response)
curl -X POST "https://api.ipulse.one/toolbox/port-scanner" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"target": "example.com", "scanType": "quick"}'
```
